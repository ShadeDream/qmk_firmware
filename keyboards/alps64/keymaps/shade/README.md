```
|Ctrl  |  A|  S|  D|  F|  G|  H|  J|  K|  L|  ;|  '| Enter  |

|-----------------------------------------------------------'
|Ctrl |Alt |GUI  |         Space          |GUI  |FN  |Ctrl  |
`-----------------------------------------------------------'
```
