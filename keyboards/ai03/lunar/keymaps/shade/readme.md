```
|---------------------------------------------------------------|
|Ctrl  |  A|  S|  D|  F|  G|  H|  J|  K|  L|  ;|  '| Enter  |PUp|

|Shift   |  Z|  X|  C|  V|  B|  N|  M|  ,|  .|  /| Shift| Up|PDn|
|---------------------------------------------------------------|
|Ctrl |Alt |GUI  |         Space         |GUI  |FN  |Lft|Dwn|Rgt|

```
